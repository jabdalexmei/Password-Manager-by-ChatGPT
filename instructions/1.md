

## ТЗ UI-POLISH-VAULT-01 — привести Vault экран к “продуктовому” виду

### Цель

Сделать Vault визуально цельным и “дорогим”:

* исправить отсутствующий слой-обёртку `vault-panel-wrapper`
* добавить правильные hover/focus состояния
* выровнять ритм: отступы, высоты, визуальные акценты
* ничего не ломать в логике и i18n

---

## 1) Добавить базовый слой панелей: `vault-panel-wrapper`

### Файл

`src/styles/screens/vault.css`

### Добавить в конец файла (или рядом с блоками Vault panels)

```css
/* Panel layout wrapper used by DataCards/Details panels */
.vault-panel-wrapper {
  display: flex;
  flex-direction: column;
  gap: 12px;
  min-height: 0; /* critical for proper inner scroll areas */
}

/* Make section headers look consistent */
.vault-section-header {
  font-size: 14px;
  line-height: 18px;
}
```

**Зачем:** это лечит “сырой” вид панели DataCards (сейчас wrapper не стилизован вообще).

## 2) Hover + focus-visible для папок и карточек (ощущение качества)

### Файл

`src/styles/screens/vault.css`

### 2.1 Data cards list items

Добавить:

```css
.vault-datacard:hover {
  border-color: rgba(59, 130, 246, 0.45); /* primary hint */
  background: var(--color-surface-2);
  transform: translateY(-1px);
  transition: transform 0.12s ease, background 0.12s ease, border-color 0.12s ease;
}

.vault-datacard:focus-visible {
  outline: 2px solid var(--color-primary);
  outline-offset: 2px;
}
```

### 2.2 Folder items

Добавить:

```css
.vault-folder:hover {
  border-color: rgba(59, 130, 246, 0.45);
  background: var(--color-surface-2);
  transform: translateY(-1px);
  transition: transform 0.12s ease, background 0.12s ease, border-color 0.12s ease;
}

.vault-folder:focus-visible {
  outline: 2px solid var(--color-primary);
  outline-offset: 2px;
}
```

## 3) Причесать поисковую строку и сайдбар-actions

### Файл

`src/styles/screens/vault.css`

### 3.1 Search input

Найти `.vault-search` и убедиться, что есть минимум:

```css
.vault-search {
  width: 100%;
  padding: 12px 12px;
  border-radius: var(--radius-md);
  border: 1px solid var(--color-border);
  background: var(--color-input-bg);
  color: var(--color-text);
}

.vault-search:focus-visible {
  outline: 2px solid var(--color-primary);
  border-color: transparent;
}
```

### 3.2 Sidebar actions spacing

Найти/добавить `.vault-sidebar-actions`:

```css
.vault-sidebar-actions {
  display: flex;
  gap: 10px;
  margin-bottom: 12px;
}

.vault-sidebar-actions .btn {
  flex: 1;
}
```
