Ниже — **ТЗ на редизайн окна Password generator** “как на After” (без кривизны), с **точными файлами** и **какой код удалять/вставлять**. Я ориентируюсь на твои скрины `Before.png` (как сейчас) и `after.png` (как должно стать).

---

## Цель редизайна (что должно получиться визуально)

1. **Убрать “полоски-разделители”**, которые режут модалку (линии под хедером и над футером).
2. Вверху: `Generated password` + input, справа **2 иконки**:

   * Copy — обычная (тёмная)
   * Regenerate — **primary**, синяя заливка (как на After)
3. Блок силы пароля:

   * строка: слева `Password strength`, справа `Strong (XXX bits)`
   * ниже **полоса силы** на всю ширину (как на After)
   * убрать отдельный текст `Strong (...)` под полосой и убрать отдельный `XXX bits` справа от полосы (как в Before)
4. `Length: N` + слайдер (на всю ширину)
5. Чекбоксы — **в один столбик** (как на After), не в grid 2 колонки
6. Футер: `Cancel` слева, `Use` справа, без разделительной линии.
7. **Primary color** (везде в приложении): сделать **более синим**, не “голубым”.

## 1) Правки разметки модалки

### Файл

`src/features/Vault/components/modals/PasswordGeneratorModal.tsx`

### 1.1 Удалить текущий StrengthBar и нижний текст

Удалить полностью компонент `StrengthBar` и использование:

**Удалить блок:**

```tsx
const StrengthBar: React.FC<{ bits: number }> = ({ bits }) => {
  const maxBits = 128;
  const t = clamp(bits / maxBits, 0, 1);
  const hue = 120 * t;
  const percentage = t * 100;
  const color = `hsl(${hue}, 80%, 45%)`;

  return (
    <div className="generator-strength">
      <div className="generator-strength-bar">
        <div className="generator-strength-fill" style={{ width: `${percentage}%`, backgroundColor: color }} />
      </div>
      <span className="generator-strength-label">{bits} bits</span>
    </div>
  );
};
```

**И удалить 2 строки в JSX:**

```tsx
<StrengthBar bits={bits} />
<div className="generator-strength-text">{t('generator.strengthLabel', { bits })}</div>
```

### 1.2 Вставить новый strength-блок “как на After”

Внутри `<div className="dialog-body generator-body">` **сразу после блока Generated password** вставить:

```tsx
<div className="generator-strength-block">
  <div className="generator-strength-header">
    <span className="generator-strength-title">{t('generator.passwordStrength')}</span>
    <span className="generator-strength-value">{t('generator.strengthLabel', { bits })}</span>
  </div>

  <div className="generator-strength-bar">
    <div
      className="generator-strength-fill"
      style={{ width: `${Math.min(100, (bits / 128) * 100)}%` }}
    />
  </div>
</div>
```

> Примечание: `bits/128` — это только масштаб полосы. Текст справа всегда показывает реальные bits.

### 1.3 Сделать кнопку regenerate primary-синей

В блоке input-actions поменять regenerate кнопку:

**Было:**

```tsx
<button className="icon-button" type="button" aria-label={t('generator.regenerate')} onClick={onRegenerate}>
  <GenerateIcon width={20} height={20} />
</button>
```

**Стало:**

```tsx
<button
  className="icon-button icon-button-primary"
  type="button"
  aria-label={t('generator.regenerate')}
  onClick={onRegenerate}
>
  <GenerateIcon width={20} height={20} />
</button>
```

Copy-кнопку не трогаем (она остаётся обычной).

### 1.4 Чекбоксы — в один столбик

Разметку чекбоксов можно оставить как есть (она нормальная), но **CSS** переведём из grid в column (см. ниже).

### 1.5 Футер — Cancel слева, Use справа

Разметку футера можно оставить, но CSS футера для generator-dialog поменять на `space-between` (см. ниже).

## 3) CSS: убрать “полоски”, перестроить layout, выровнять футер

#### 3.1 Убрать линии только в Password generator (не ломая другие диалоги)

```css

  padding-bottom: 10px;
}

.generator-dialog .dialog-footer {
  border-top: none;
  justify-content: space-between;
  padding-top: 10px;
```

#### 3.2 Сделать generator-options вертикальным списком

Заменить текущий `.generator-options`:

**Стало:**

```css
}
```
