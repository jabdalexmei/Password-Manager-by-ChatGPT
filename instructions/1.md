

## TECH SPEC (diff-like) — Create profile: remove inner panel border, unify surface, neutral inputs (no blue)

### Scope

Only “Create profile” screen.
No behavior changes. No component structure changes (optional: CSS-only fix).

### Target state (visual)

* The form should NOT look like it sits inside a separate bordered/dashed panel.
* Card background remains the same as **Select profile** (`--color-surface`).
* Inputs should not use the blue-ish `--color-input-bg` surface. They must use the same subtle neutral tint used by Select profile empty-state: `background: var(--surface-tint-02);`
* Inputs have **no border/outline ring in normal state** (no visible stroke).
* On keyboard focus, show a **neutral (non-blue)** focus indicator (still visible).

> Accessibility note: if we remove default outlines/borders, we must provide a visible focus style for keyboard users.
