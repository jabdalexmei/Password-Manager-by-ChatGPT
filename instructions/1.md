
## TS (English): Make Expiry date and CVC/CVV fields the same compact width in BankCard Details

### Goal

In **BankCard Details** panel, `Expiry date` and `CVC/CVV` value boxes must have the **same compact width**, while keeping the normal vertical field layout.

### File

`src/styles/screens/vault.css` 

### Change

Add a shared rule (or replace existing separate rules) so both selectors use the same `max-width`.

#### If you already have separate rules like:

```css
.bankcard-detail-card .bankcard-compact-expiry .detail-value-box { max-width: 520px; }
.bankcard-detail-card .bankcard-compact-cvc .detail-value-box { max-width: 360px; }
```

#### Replace them with a single rule (exact):

```css
/* Bank card details: Expiry and CVC must be same compact width */
.bankcard-detail-card .bankcard-compact-expiry .detail-value-box,
.bankcard-detail-card .bankcard-compact-cvc .detail-value-box {
  max-width: 360px;
}
```

*(Число 360px — ориентир по твоему скрину: CVC уже выглядит “правильно”, значит подгоняем Expiry под него. Если хочешь чуть шире — поставь 380/400.)*

### Acceptance criteria

* Expiry date value box width == CVC/CVV value box width (visually identical).
* Both remain shorter than full-width fields.
* CVC still fits Copy/Eye buttons without wrapping.
